## ✨ 核心功能  
  
### 政治备考板块  
- 考点结构化梳理：按马原、毛中特、史纲、思修、时政五大模块分类，**红色标注必背考点**、**黄色标注高频考点**、**蓝色标注理解性考点**，重点一目了然  
- 资料格式多样化：支持HTML在线阅读（带考点高亮+注释）、Anki卡片导出（含正反面模板）、PDF高清下载（可直接打印）  
  
### 通用功能  
- 资料搜索：支持按关键词、考点类型、难度等级、年份筛选内容  
- 自定义收藏：收藏重点资料、标记复习进度，生成个人专属复习清单  
- 格式导出：支持Anki（.apkg）、PDF、HTML、Markdown四种格式导出  
- 进度记录：自动记录学习时长、已掌握考点，未完成内容弹窗提醒  
- 夜间模式：支持切换明暗主题，适配长时间学习场景  
  
### 高级功能  
- **考点筛选**：通过URL参数精准筛选考点，示例：  
  ```  
  {{ORIGIN_URL}}/politics/mao?highlight=必背&type=简答题  
  {{ORIGIN_URL}}/mtipro/words?category=时政热词&year=2025  
  {{ORIGIN_URL}}/mtipro/essay?type=应用文&topic=通知  
  ```  
- **Anki自定义生成**：支持选择考点范围、卡片样式、是否包含解析，示例：  
  ```  
  {{ORIGIN_URL}}/export/anki?subject=politics&module=马原&style=简约版&include解析=true  
  {{ORIGIN_URL}}/export/anki?subject=mtipro&module=词汇&style=图文版  
  ```  
- **进度同步**：登录后可同步多设备学习进度、收藏夹、错题本、笔记内容  
  
## 🚀 部署指南  
  
### 一键部署（零基础首选）  
- [Railway](https://railway.app/)：直接关联GitHub仓库，自动部署无需手动配置  
- [GitHub Pages快速部署](https://github.com/1690291242-bot/169/wiki/GitHub-Pages%E9%83%A8%E7%BD%B2%E6%96%B9%E6%B3%95)：专为零基础设计，3步完成部署  
  
### Docker Compose 部署（稳定可靠）  
```sh  
# 1. 克隆仓库到本地  
git clone https://github.com/1690291242-bot/169.git && cd 169  
  
# 2. 启动服务（自动安装依赖、配置环境，默认端口3000）  
docker-compose up -d  
```  
> 参考配置文件：[docker-compose.yml](./docker-compose.yml)  
  
### 本地部署（适合有基础的用户）  
```sh  
# 1. 安装依赖（需提前安装Node.js 18+）  
npm install  
  
# 2. 构建项目  
npm run build  
  
# 3. 启动服务（默认端口3000，可在.env文件修改）  
npm run start  
  
# 4. 开发模式启动（实时热更新，用于修改内容）  
npm run dev  
```  
  
### 环境变量配置（可选，根目录创建.env文件）  
| 变量名               | 说明                                  | 默认值                  |  
| -------------------- | ------------------------------------- | ----------------------- |  
| `PORT`               | 服务运行端口                          | 3000                    |  
| `DATABASE_TYPE`      | 数据库类型（支持sqlite/mysql）        | sqlite（无需额外配置）  |  
| `AUTH_ENABLE`        | 是否启用登录功能（同步进度需开启）    | false（默认关闭）       |  
| `HIGHLIGHT_STYLE`    | 考点高亮样式（red/yellow/blue）       | red=必背，yellow=高频，blue=理解 |  
| `UPDATE_CRON`        | 资料自动更新Cron表达式                | 0 3 * * *（每日凌晨3点）|  
  
## 📱 使用说明  
  
### 1. 基础使用（无需登录）  
1. 访问网站首页（本地部署：http://localhost:3000），选择「政治备考」或「翻译硕士」板块  
2. 按模块浏览资料（如：政治→马原→唯物论→物质与意识）  
3. 高亮部分为重点考点，点击「查看解析」可查看详细注释  
4. 点击页面右侧「导出」按钮，选择需要的格式（Anki/PDF/HTML/Markdown）下载使用  
  
### 2. 进阶使用（登录后）  
1. 注册/登录账号（支持GitHub账号快捷登录）  
2. 收藏重点资料：点击资料卡片右上角「⭐」添加到个人收藏夹  
3. 标记学习进度：阅读后点击「已掌握」「需复习」「未理解」记录状态  
4. 自定义设置：在「个人中心」调整高亮颜色、默认导出格式、是否开启夜间模式  
5. 同步多设备：登录相同账号，自动同步所有学习数据  
  
### 3. 资料更新与补充  
- 自动更新：后台每日凌晨同步最新时政、大纲变化、新增真题  
- 手动上传：支持上传个人笔记（Markdown格式），自动整合到对应模块  
- 提醒设置：可开启邮件提醒，接收资料更新、复习计划到期通知
